Release Notes for WESNet Designs By Invitation v 01.00.02

Minimum DotNetNuke Framework Version: 07.03.03

Issues Fixed in This Release:

  • Null Reference Exception occurs when invitations are emailed from within scheduled job if advanced url rewriting is NOT enabled (urlFormat="advanced") in web.config. New installs of DNN 7.2 and later will by default use advanced url rewriting. However sites upgraded from earlier DNN versions will not have advanced url rewriting enabled unless web.config has been manually modified. The DNN core's BasicFriendlyUrlProvider.GetFriendlyAlias method which is called from NavigateUrl method cannot be called from within a scheduled job as no HttpContext is available. This longstanding core issue was corrected in the AdvancedUrlProvider.GetFriendlyAlias method but never back ported to the BasicFriendlyUrlProvider. Applied workaround by building the Join Now and Decline URL's by hand rather than calling NavigateUrl from with a scheduled job.
  • Null Reference Exception occurs in core NotificationsController during enumeration of recipient UserInfo objects when invitations are manually created. Updated UpdateInvitationStatus stored procedure to perform left outer join rather than inner join on DNN Tabs table. Issue is indirectly triggered by change in DNN 7.4.1 and later versions, from storing the default (none) redirect value on login TabID in PortalSettings as null to no entry in PortalSettings table when default.
  • When recipient of invitation clicks on emailed "Join Now" or "Decline" links, he receives a message that the link is invalid and is not able to proceed with invitation processing. Modified join type on Tabs table in GetInvitationByPortalAndRSVPCode stored procedure.
  • Fixed several SQL stored proceedures having Tabs table join issues as above.
  • System.ArgumentException is thrown with message "Action must be one of: System.Linq.Enumerable" during resending of invitations by scheduled job when inviting user or administrator notification of resent invitations is enabled. Fixed typing error in list of valid action strings.
  • When invitation acceptance processor form is configured to require entry of temporary password contained in email invitation, the temporary password was already filled in upon initial display of the form.

Enhancements and Other Changes:

  • None

Date of Release: July 18, 2015
